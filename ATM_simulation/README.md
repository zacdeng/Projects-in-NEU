### C++模拟银行ATM机

* 功能：创建账户、登录账户、自动存取款、修改密码、转账、查询存取款&转账记录功能
* 存储方式：.csv文件格式存储用户信息、存取款&转账信息
* IDE：本代码为Clion工程文件，需要时直接git clone至相关clion工程目录即可使用
